# Campus Safety & Incident Reporting System (CSIRS) - 2026 TVET Fair

A secure, role-based, anonymous incident reporting platform for TVET institutions.

## Tech Stack (Monorepo with Turborepo + npm)

- **Frontend**: React 18 + TypeScript + Vite + Redux Toolkit + RTK Query + Tailwind + shadcn/ui
- **Backend**: Node.js + Express + TypeScript + Prisma + PostgreSQL
- **Shared**: TypeScript types, Zod schemas, utilities
- **Monorepo**: Turborepo (npm workspaces)
